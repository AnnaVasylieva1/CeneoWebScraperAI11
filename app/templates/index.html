Project's purpose:
This web scraper is made for extraction of opinions from a product page of Ceneo.pl and display them using some additional statistical tools
The application waits for product code that the user provides, and then it checks the responce from the server. Is response is OK, it continues to extract the whole site DOM, after extraction it takes all opinions and their parts and puts them into a JSON file.
Libraries Used:

  • Pandas
  • Numpy
  • Requests
  • GoogleTranslator
  • JSON
  • BeautifulSoup
  • Matplotlib
